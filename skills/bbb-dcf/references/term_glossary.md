# Türkçe Terim Sözlüğü — DCF Değerleme

**Kural:** Tüm çıktılarda **İLK KULLANILDA** kısaltma + Türkçe parantez. Final çıktı formatında **sadece Türkçe** kullanılmalı.

| Kısaltma | Türkçe Karşılığı |
|----------|-----------------|
| D/E | Borç/Özkaynak Oranı |
| WACC | Ağırlıklı Ortalama Sermaye Maliyeti |
| Rf | Risksiz Faiz Oranı |
| ERP | Özkaynak Risk Primi |
| CRP | Ülke Risk Primi |
| β / Beta | Beta Katsayısı |
| Ke | Özkaynak Maliyeti |
| Kd | Borç Maliyeti |
| EBIT | Faiz ve Vergi Öncesi Kâr |
| EBITDA | Faiz, Vergi, Amortisman Öncesi Kâr |
| FCFF | Firmaya Serbest Nakit Akışı |
| D&A | Amortisman ve İtfa Payları |
| Capex | Sermaye Harcaması |
| ΔWC | İşletme Sermayesi Değişimi |
| HAO | Halka Açıklık Oranı |
| KGOP | Kontrol Gücü Olmayan Paylar (Azınlık Payları) |
| NAV | Net Aktif Değer |
| SOTP | Parçaların Toplamı Değerlemesi |
| DCF | İndirgenmiş Nakit Akışı |
| TTM | Son 12 Ay |
| ICR | Faiz Karşılama Oranı |
| ROIC | Yatırılan Sermaye Getirisi |
| Terminal Value | Son Değer |
| Fade Period | Yakınsama Dönemi |
