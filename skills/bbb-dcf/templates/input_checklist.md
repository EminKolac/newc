## Aşama 2: Piyasa Verileri

## Aşama 3: Referans Verileri

- [ ] Sektör unlevered beta (Damodaran Global)

- [ ] Sektör P/E, EV/EBITDA (relative check için)

## Aşama 4: Kullanıcı Kararları

- [ ] Gelir büyümesi yıl 1
- [ ] Gelir büyümesi yıl 2-5
- [ ] Hedef EBIT marjı
- [ ] Marj yakınsama yılı (1-10)
- [ ] Sales/Capital ratio (1-5)
- [ ] Sales/Capital ratio (6-10)
- [ ] Terminal WACC override? (Evet → değer)
- [ ] Terminal ROC override? (Evet → değer)
- [ ] Failure probability? (Evet → %)
- [ ] R&D capitalize? (BIST'te genelde Hayır)

- [ ] Döngüsel mi? → EBIT normalleştirme gerekli mi?
